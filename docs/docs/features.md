# Features

## Visual Assessment Agent

Detects visible damage using computer vision.

## Multilingual Diagnostic Agent

Collects device symptoms through voice interaction.

## Circular Intelligence Engine

Determines the best next life for the device.

## Sustainability Impact Engine

Calculates CO₂ avoided and e-waste prevented.

## Repair Marketplace

Connects users with technicians.

## Impact Ledger

Tracks collective sustainability impact.
